ASCII

ASCII is the American Standard Code for Information Interchange and uses 7 bit numbers to represent characters. This gives us 27 or 128 possible characters. 33 characters are reserved as control characters like RETURN or TAB leaving 95 printable characters starting with the SPACE character.

As you can see from the table ASCII is used for text only files such as those opened by Windows Notepad.
